# Hospital Patient Record System

## 📖 Overview
This project is a hospital patient record management system developed in Java.  
It focuses on efficient data handling using fundamental data structures and algorithms.

## 🚀 Features
- Patient registration and record management
- Data storage using LinkedList, Queue, and HashMap
- Sorting algorithms (Merge Sort, Quick Sort, Heap Sort)
- Efficient data retrieval and organization

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Data Structures & Algorithms
